### 安装

```sh
npm install web-win-vue --save
```

### 注意

以下组件目前不为积极开发状态:

- webwin-tile
- webwin-button

以下组件正在开发中，部分内容随时可能变动:

- webwin-topappbar

Github Release 目前只会在 稳定版本 发布后推送。

### 使用

导入:

```sh
import { winbutton } from 'web-win-vue'
import 'web-win-vue/web-win-vue.css'
```
